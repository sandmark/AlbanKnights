======
使い方
======
起動すると
AlbanKnights(1):
のような無機質なプロンプトが表示されます。
基本的にはここにコマンドを打ち込んで操作します。
一度打ち込んだコマンドは、↑キーで履歴を呼び出すことができます。

(1) に特に意味はありません。
コマンドを打ち込むたびに (2),(3)... のように増えるだけで、
プログラムを起動してからどれくらい経ったかの目安にしてください。

終了する場合は、 "quit","exit","q",":q" のいずれかを入力してください。
どの終了コマンドを使っても同じですが、
表示されるメッセージについては後述の説明を参考にしてください。

============
適当な使い方
============
AlbanKnights: kaour 1
のように打ち込むと、カオルの好感度番号が 1 のときの
キーワードが３つ表示されます。

例:
AlbanKnights(1): kaour 1
【カオル】    (1)    遊び -> 遊び -> 訓練
AlbanKnights(2):

また、カタカナやローマ字、省略記法にも対応しています。

例:
AlbanKnights(2): kaoru 4
【カオル】    (4)    料理 -> 遊び -> 恋愛
AlbanKnights(3): k 7
【カオル】    (7)    訓練 -> 料理 -> 恋愛
AlbanKnights(4): カオル 10
【カオル】    (10)   料理 -> 任務 -> 任務

NPCコマンドの一覧は以下のようになっています。
右側の欄はキャラクターに対応する英名、ローマ字、省略形です。

+------------+--------------------+
| カオル     | kaour, kaoru, k    |
| アイリース | eirlys, airi-su, a |
| ダイ       | dai, d             |
| エルシィ   | elsie, erusii, e   |
+------------+--------------------+

============
普通の使い方
============
"NPC名 <番号>" のように全員分打ち込むのは面倒な上に、
次の番号をいちいち計算しなければならない現実にムカついたので
まとめて管理してくれる表を実装しました。

AlbanKnights: l
【ダイ】        (不明)
【アイリース】  (不明)
【カオル】      (不明)
【エルシィ】    (不明)

"l" は "list" の略で、 "ls","show" などでも呼び出すことができます。
(不明) となっている場所は番号が入る部分ですが、
プログラム開始時には何もセットされていません。

------
セット
------

例として、ダイに 1 をセットしてみます。
NPCの番号をセットするには "set" コマンドを使います。

AlbanKnights: set dai 1
Albanknights: l
【ダイ】        (1)    遊び -> 恋愛 -> 遊び
【アイリース】  (不明)
【カオル】      (不明)
【エルシィ】    (不明)

"set" コマンドは番号をセットするだけで何も表示しませんが、
"list" コマンドで確認してみると、ダイに 1 がセットされ、
１番目から数えたキーワードが３つ表示されているのがわかります。

また、先述の通り "dai" は "d" でも代用できるため
"set d 1" としても同じようにセットできます。
さらに "set" の省略形として "s" が用意されているので、
"s d 1" のように暗号っぽいコマンドでも大丈夫です。
慣れてきたり、トイレに行きたいときなど急いでいるときに使ってください。

他のNPCに番号をセットするときも同じ方法が使えます。

AlbanKnights: set kaour 23
AlbanKnights: set a 96
AlbanKnights: s erusii 57
AlbanKnights: l
【ダイ】        (1)    遊び -> 恋愛 -> 遊び
【アイリース】  (96)   恋愛 -> 料理 -> 任務
【カオル】      (23)   任務 -> ファッション -> 遊び
【エルシィ】    (57)   遊び -> 遊び -> ファッション

セットする順番は関係ありません。
「ダイが１じゃなくて７かも」と思ったら、

AlbanKnights: set dai 7
AlbanKnights: l
【ダイ】        (7)    遊び -> 遊び -> 遊び
【アイリース】  (96)   恋愛 -> 料理 -> 任務
【カオル】      (23)   任務 -> ファッション -> 遊び
【エルシィ】    (57)   遊び -> 遊び -> ファッション

このようにしていつでも変更できます。

----------
アンセット
----------
対応するコマンドに "unset","u" があり、
"unset dai" のように呼び出すことで (不明) 状態に戻せます。

AlbanKnights: u dai
AlbanKnights: l
【ダイ】        (不明)
【アイリース】  (96)   恋愛 -> 料理 -> 任務
【カオル】      (23)   任務 -> ファッション -> 遊び
【エルシィ】    (57)   遊び -> 遊び -> ファッション

あんまり使う機会はないかもしれませんが、
イベント会話などでこんがらがったときにどうぞ。

------------
アップデート
------------
表を用意したものの、いちいち計算してセットし直すのは面倒なので
"update" コマンドを用意しました。 "up","next","x" でも呼び出せます。

AlbanKnights: l
【ダイ】        (不明)
【アイリース】  (96)   恋愛 -> 料理 -> 任務
【カオル】      (23)   任務 -> ファッション -> 遊び
【エルシィ】    (57)   遊び -> 遊び -> ファッション

表がこのようになっているときに "update" を行うと、
番号が再計算されて新たにセットされます。
(不明) となっている部分は変更されません。

AlbanKnights: update
AlbanKnights: l
【ダイ】        (不明)
【アイリース】  (2)    遊び -> 料理 -> 遊び
【カオル】      (26)   任務 -> ファッション -> 訓練
【エルシィ】    (60)   恋愛 -> 遊び -> 遊び

============
大まかな流れ
============
それぞれのNPCに番号をセットしてから "l" でキーワードを表示し、
その後は延々と "update" or "x" からの "l" で次のキーワードを確認する、
というワークフローになるかと思います。
基本的には終了させずに常時起動させておくコンセプトです。

NPCに番号をまとめてセットする方法は（まだ）ありませんが、
起動直後に行うだけなのでそれほど手間にはならないかと。

なお、終了時には番号をどこかにメモれという旨のメッセージが表示されます。
うっかりウィンドウを×ボタンで閉じてしまうと表示されないので、
必ず "exit" コマンドで終了するようにしてください。
